# QA Bridge Mix

Este repositório contém a documentação de Quality Assurance (QA) do projeto **Bridge Mix**, um gerador de playlists baseado em IA.

O objetivo deste projeto é estruturar qualidade desde a fase inicial do produto, incluindo entendimento do negócio, levantamento de requisitos, análise de riscos, planejamento de testes e elaboração de casos de teste.

Este repositório faz parte do meu portfólio de QA.
